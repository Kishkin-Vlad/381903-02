* support for variables of the form <letters><number> (example: 'line1', 'vec21', 'dfgrkvgmfr2341234')
* support for multiplication <experession><number> or <expression><variable> etc (example: '23(1 + a)', '(12 + 4) / (2 + 2)')
* support for many spaces (example: '   2  +    -12  ^2   ')
* support for unary minus (example: '2 + -1', '2 - ---1')
* operators: +, -, *, /, ^
